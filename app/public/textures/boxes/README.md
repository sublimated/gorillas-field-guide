# Hand-drawn box frames

Each is one of the user's hand-drawn box outlines, tinted to ink (#2c2013) so the
lines read as ink on parchment. Use as a `background-image` with
`background-size: 100% 100%`. Pick the box whose aspect ratio is CLOSEST to the
element, then size the element toward that ratio so the hand-drawn line isn't
distorted (don't stretch a square box across a wide pill).

| file          | ratio (w/h) | shape         | good for |
|---------------|-------------|---------------|----------|
| box-03.svg    | 0.97        | square        | square chips |
| square.svg    | 1.03        | square (big)  | large square containers |
| box-01.svg    | 1.18        | squarish      | sigil cells / cards |
| box-11.svg    | 1.75        | medium-wide   | syllable chips, small panels |
| box-02.svg    | 1.89        | wide          | medium panels |
| box-05.svg    | 1.93        | wide          | medium panels |
| box-08.svg    | 2.41        | wide          | stat rows |
| box-07.svg    | 2.45        | wide          | tabs |
| box-10.svg    | 2.68        | wide pill     | version tabs |
| box-06.svg    | 2.85        | wide pill     | toggles, buttons, level/school tabs |
| box-04.svg    | 3.90        | long          | dropdowns/selects |
| box-09.svg    | 0.41        | tall          | vertical elements |
| search-bar.svg| 14.57       | long bar      | the search input |
